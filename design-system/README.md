# Khelo Bharat Design System

Source of truth for the Khelo Bharat platform's visual language, synced from the
website repo (`assets/css/site.css` + `assets/js/site.js` components).

- **Foundations**: colour tokens (brand, pillar identities P1–P6, integration-class
  colours), typography scale (Fraunces display / Inter body), evidence chips.
- **Components**: stat tiles, journey flow tiles, bar charts, hub-and-spoke
  integration maps, pillar tiles & module cards, navigation chrome.
- **Screens**: home hero, module page hero — the two key dark-band screens.

Design rules: institutional, calm, evidence-first. Every claim carries an evidence
chip; proposed designs are never presented as existing systems. Wireframes are
illustrative concepts, not built product.
